텍스트 관련 태그

abbr : abbreviation(= 줄임말)의 줄임말. title 속성도 함께 사용할 수 있다.
IoT

cite : citation(= 인용)의 줄임말. 참고 내용을 표시한다.
내가 경험한 가장 흥미진진한 일은 누군가를 만나는 일이다 - 영화, '비포선셋' 중

code : source code를 표시한다.
std::cout << "Hello, world" << std::endl;

small : 부가 정보처럼 작게 표시해도 되는 텍스트를 표시한다.
정보 : 이 텍스트는 읽을 필요가 없다.

sub : 아래 첨자를 표시한다.
물의 화학식은 H2O이다.

sup : 위 첨자를 표시한다.
지수함수적 시간복잡도는 O(2n)과 같이 표현한다.

s : 취소선을 표시한다. slice의 줄임말인 듯하다.
농담입니다.

u : underbar의 줄임말이다. 밑줄을 표시한다.
여기부터 밑줄 ~~

ins : insert의 줄임말이다. 언뜻 보면 밑줄 표시와 같지만, 새로운 내용을 삽입함을 나타낸다.
새로운 내용을 삽입합니다.

del : delete의 줄임말이다. 언뜻 보면 취소선 표시와 같지만, 기존 내용을 삭제함을 나타낸다.
기존 내용을 삭제합니다.
